# 📊 E-commerce Sales Analysis

## 📌 Project Overview

This project analyzes an e-commerce sales dataset to uncover key business insights related to sales performance, products, regions, and customer behavior.

---

## 🧰 Tools Used

* Python (Pandas)
* Microsoft Excel
* Tableau

---

## 📂 Dataset

* Source: Kaggle
* Rows: 9,800
* Columns: 18

---

## 🧹 Data Cleaning

* Converted date columns to DateTime format
* Found 11 missing values in Postal Code (no major impact on analysis)

---

## 📊 Key Analysis

### 🔹 Sales by Year

* 2015: 479,856
* 2016: 459,436
* 2017: 600,192
* 2018: 722,052

📈 Sales increased significantly after 2016.

---

### 🔹 Top Products

* Canon imageCLASS 2200 Advanced Copier
* Fellowes PB500 Binding Machine
* Cisco TelePresence System

💡 High-value products generate the most revenue.

---

### 🔹 Sales by Region

* West (Highest)
* East
* Central
* South

🌍 West region leads in sales performance.

---

### 🔹 Sales by Category

* Technology (Top)
* Furniture
* Office Supplies

💡 Technology is the main revenue driver.

---

### 🔹 Shipping Mode

* Standard Class (Most used)

🚚 Customers prefer cost-effective shipping.

---

## 📌 Key Insights

* Strong growth from 2016 to 2018
* Technology dominates revenue
* West region is the best-performing market
